# Product Overview

BrimApp is a cross-platform note-taking application with AI-powered features built with React Native, Expo, and Supabase.

## Core Features

- **Note Management**: Create, edit, archive, and delete notes with rich text support
- **AI Integration**: Grammar fixing and text rephrasing using Google's Gemini API
- **Voice Transcription**: Audio-to-text conversion for voice notes
- **Real-time Sync**: Cross-device synchronization via Supabase
- **Customization**: Light/dark themes and accent color options
- **Internationalization**: Multi-language support with i18next
- **Authentication**: Secure user authentication with Apple Sign-In support

## Target Platforms

- iOS (primary)
- Android
- Web (via Expo Web)

## Key User Flows

1. **Authentication**: Users sign in via Apple ID or phone number
2. **Note Creation**: Create notes via text input or voice recording
3. **AI Enhancement**: Use AI to fix grammar or rephrase content
4. **Organization**: Archive or trash notes for organization
5. **Sync**: Notes automatically sync across all user devices
